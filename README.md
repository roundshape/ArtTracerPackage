# ArtTracer

Adobe Illustrator 用 **テンプレート流し込みプラグイン**。Illustrator のアートを `.artx`（テンプレート XML）として保存し、CSV データを差し込むことで複数バリエーションを一気に展開できます。

`.artx` は AI（LLM）が直接生成できるよう設計された人間にも読めるテキスト形式の XML なので、**「ChatGPT / Claude にデザインを依頼 → ArtTracer で Illustrator に流し込む」** というワークフローが成り立ちます。

---

## できること

### 1. テンプレート保存と再利用

Illustrator で作成したアートを `.artx` 形式で保存。後から同じテンプレートを呼び出して、好きな位置にクリック 1 回で描画できます。

### 2. CSV 流し込みで一括バリエーション生成

テキスト内に `{店名}` `{電話番号}` のような **プレースホルダ** を埋め込んで `.artx` を保存しておけば、CSV を渡すだけで行ごとにデータを差し替えた複数バリエーションを一発展開できます。

**用途例**：店舗別のショップカード／商品別の値札／個人別の名刺／講座別のチラシ ／ 季節販促物の差分展開 など

### 3. AI による .artx の直接生成 ★

`.artx` の仕様 (`ai_docs/artx-format.md`) は **LLM が読んでそのまま `.artx` を出力できる** よう設計されています。

#### 典型的なワークフロー

1. ChatGPT / Claude / Gemini など LLM のチャットに `ai_docs/artx-format.md` を貼り付け（または添付）
2. 「**A6 サイズの美容室ショップカードを作って。`{name}` `{phone}` `{address}` のプレースホルダを入れて**」のように依頼
3. LLM が `mycard.artx` を出力 → 任意のフォルダに保存
4. Illustrator で **ウィンドウ → ArtTracer** からパネルを開き、`mycard.artx` を選択
5. パネルから CSV（店舗一覧）を読み込み
6. 描画ツールでクリックすると、各行のデータが差し込まれたカードが連続描画される

AI に絵筆を持たせて、Illustrator のアートとして実体化させる流れです。Illustrator 上でいつでも手作業でリタッチできるので、AI が完璧でなくても問題ありません。

## サポートする要素

### アート要素
- パス（オープン / クローズ）、複合パス
- グループ
- テキスト（ポイント・エリア・パス上）
  - 文字属性: フォント・サイズ・色・トラッキング・水平/垂直スケール・ベースラインシフト・カーニング・行送り
  - エリアテキストの **溢れ自動調整**
- 配置画像（PNG / JPEG）

### 塗り・線
- 単色（RGB / CMYK / Gray）
- リニア / ラジアルグラデーション
- 線幅・破線パターン

### その他
- クリッピングマスク（パス・複合パス）
- 描画ツールの **繰り返しモード**（Esc / Option で抜ける）
- パネル内 CSV ページャ（`<` `>` で行を循環移動）

## Finder 連携

同梱の **`ArtTracerHelper.app`** を `/Applications` にコピーすると、`.artx` ファイルが Finder で次のように扱われます：

- 専用アイコン表示
- サムネイル画像（`.artx` に保存時に埋め込まれる PNG プレビュー）
- Quick Look プレビュー（スペースキーで実物大プレビュー）

---

## インストール

**対応プラットフォーム**: macOS のみ（Adobe Illustrator 2022 以降）。Windows 版は現時点で未提供です。

dmg 内の **`Install Guide.md`** を参照してください。

要点だけ：

1. `ArtTracerHelper.app` を `/Applications` にドラッグ
2. `ArtTracer.aip` を Illustrator の `Plug-ins.localized/` フォルダにコピー
3. Illustrator を再起動
