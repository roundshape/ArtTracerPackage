# ArtTracer 使い方ガイド

ArtTracer は、Illustrator のアートを **テンプレート（`.artx`）** として保存し、**CSV データの差し込み**や **AI** で一気にバリエーション展開できるプラグインです。このガイドでは、最短で価値を出すための使い方を紹介します。

> インストールは [InstallGuide.md](InstallGuide.md) を参照してください。

---

## 1. これで何ができる？

- 作ったデザインを**テンプレート化**して、いつでもクリック 1 回で再利用
- 名刺・ショップカード・値札などを、**CSV で人数分・店舗分を一括生成**
- **パネル内の AI チャット**に「〇〇を作って」と頼むだけでデザインを生成
- 使い慣れた **ChatGPT / Claude** で作った `.artx` を取り込む

「1 枚ずつ手作業」から「データで一気に」へ。これが ArtTracer です。

---

## 2. まず使ってみる（テンプレ保存 → 描画）

1. **ウィンドウ → ArtTracer** でパネルを開く
2. Illustrator で何かアートを作って**選択**する
3. パネルの**保存**から `.artx` として保存する
   - 保存時のダイアログでは、サムネを見ながら**キーワード**（季節・イベントなど）を付けられます。後で AI がテンプレを探しやすくなります。
4. 保存したテンプレートを選び、**描画ツールでアートボードをクリック**すると、その位置にテンプレートが描画されます
   - **繰り返しモード**: クリックを続けると連続で配置できます。**Esc / Option** で抜けます。

---

## 3. CSV で一気に量産 ★

テンプレートのテキストに **プレースホルダ**（`{name}` `{phone}` のような `{ }` 囲み）を書いておくのがコツです。

1. テキスト内に `{店名}` `{電話番号}` などを入れて `.artx` を保存
2. パネルから **CSV を読み込む**（CSV の列名がプレースホルダ名に対応します）
3. 描画ツールでクリックすると、**CSV の各行のデータが差し込まれたもの**が連続で描画されます
4. パネルの **`<` `>`** で対象の行を前後に切り替えられます

→ 店舗別カード・商品別値札・個人別名刺などが、データ 1 つで一気に揃います。

---

## 4. AI に作ってもらう（パネル内チャット）★

外部サービスを行き来せず、**Illustrator の中で AI に直接依頼**できます。

1. （初回）**設定**で AI プロバイダ（**Claude / OpenAI**）と API キーを入力する
2. チャットに「**A6 の美容室カードを作って**」のように依頼する
3. AI が `.artx` を生成し、そのまま Illustrator に描画されます（生成過程はストリーミング表示）
4. 既存テンプレートを下敷きに「**色を変えて**」「**見出しを大きく**」のような改変もできます

便利なコマンド: `/clear`（履歴クリア）`/model`（モデル切替）`/retry`（やり直し）`/help`

生成後は手作業でいくらでもリタッチできるので、AI が完璧でなくても大丈夫です。

---

## 5. 外部 LLM（ChatGPT / Claude）で作る

内蔵チャットを使わず、使い慣れた LLM で作る方法もあります。

1. LLM のチャットに **[`ai_docs/artx-format.md`](ai_docs/artx-format.md)** を貼り付け／添付する
2. 「`{name}` `{phone}` を入れた A6 ショップカードを作って」のように依頼する
3. 出力された `.artx` を保存 → パネルで選んで描画（CSV 差し込みも可）

`.artx` は LLM が読んで書けるよう設計されているので、**AI とデザインの相性が良い**のが ArtTracer の特徴です。

---

## 6. Finder 連携（おまけ）

同梱の **`ArtTracerHelper.app`** を `/Applications` に入れると、`.artx` が Finder で：

- 専用アイコンで表示される
- **サムネイル**でひと目で中身が分かる
- **スペースキーで Quick Look** プレビューできる

たくさんの `.artx` を管理するときに効きます。

---

## 7. 困ったとき（FAQ）

- **パネルが出ない** → ウィンドウメニューに「ArtTracer」があるか確認。無ければインストールと Illustrator の再起動を確認してください（[InstallGuide.md](InstallGuide.md)）。
- **CSV のデータが入らない** → テキストのプレースホルダ名（`{name}` 等）と CSV の列名が一致しているか確認してください。
- **AI チャットが応答しない** → 設定の API キーとプロバイダ選択を確認してください。
- **`.artx` のサムネが出ない** → `ArtTracerHelper.app` を `/Applications` に入れ、一度ログアウト／再起動すると反映されることがあります。
